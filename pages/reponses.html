TP6 : Création et manipulation d’objets
1) Dans un objet, les attributs sont les informations/données que l’on rentre dans les champs. Les méthodes sont les actions que cet objet peut faire.
2) La fonction permettant d'instancier un objet est : 'this'.
3) Les deux façons d'accéder aux champs sont :
- nomVariable.attribut (Ex: p.age)
- nomVariable['attribut'] (Ex: p['age'])

TP9 : formatage de données en JSON
1) Les 2 espaces de stockage dans le navigateur sont: Local storage et Session storage.
2) La différence entre ces deux espaces est que pour le Local Storage, il n’y a pas d’expiration sur les données alors que pour le Session Storage, les données sont effacées systématiquement à la fermeture du navigateur.
3) Le seul type de données que l'on peut stocker dans le local Storage est les tableaux associatifs.
4) Notre modèle ne correspond pas à un tableau associatif.
5) Le lien entre l'utilisation du format JSON et l'utilisation de local storage est que nous pouvons stocker les données au format JSON dans le local storage.
